# 🤖 Anand AI - Personal AI Assistant

## 📌 Overview
Anand AI is a personal AI assistant built with Python and Google Gemini. It helps users chat with AI, analyze Excel and PDF files, generate SQL queries, and assist with data analysis.

## 🚀 Features (Version 1)
- AI Chat
- PDF Summarization
- Excel & CSV Analysis
- SQL Query Assistant
- Chat History
- Desktop Application (.exe)

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- Pandas
- SQLite
- PyInstaller
- Git & GitHub
